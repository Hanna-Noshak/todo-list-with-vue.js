# Todo List (Vue 3 + Vite)

A simple RTL Persian todo list built with Vue 3 (SFC) and Vite.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features
- Add / remove / toggle todos
- Filter: all, completed, uncompleted
- Persist in localStorage
- RTL layout, Persian date, Vazirmatn font
